Create Stop Motion Animations
Tutors

The workshop has been devised to engage students in a fun and creative activity where they produce a short stop motion animation video (10-20 secs) and post it to YouTube. The students thus learn how to use contemporary digital technologies whilst engaged in the construction of digital objects. Click below for some hints on how to use this workshop.

Workshop Organisation

This activity can be used in a single class (allow at least 2 hours) or over a number of classes. The students may work independently or collaboratively, but are strongly encouraged to help each other and exchange ideas. If time permits the students may make a number of different videos - they will gain more knowledge and confidence as they iterate through the process.

Resources Needed

  • Digital cameras with connectors - these can be shared between students.
  • Tripods, if available - small, simple ones are all that's needed.
  • Computers - these notes have been prepared for Windows 7 and using Movie Maker, but any video editing tool can be used.
  • Audio resources - it is useful to have a number of songs in mp3 format as students may find this task too time consuming.
  • Props - see next item!

Suggestions for props

Encourage students to come up with their own ideas, but some handy ideas are:

  • Glasses filling and emptying of water. Food dye can also be deployed!
  • Playing cards - build a house of cards and knock over.
  • Dominoes - lay out and knock over.
  • Plate of biscuits - to be eaten during video!
  • Pens and pencils in a container - getting tidied up.

Coloured items show up well in the end-product, especially if offset against a plain background.

Copyright Issues

Ownership of the video rests with the creator. Videos posted onto YouTube will be analysed by YouTube and music segments may be identified and attributed to the music artist. Using music segments in this way is allowed by YouTube as the videos (for the most part) only use music segments of approx 20 secs and the videos have no commercial value.

Step 2 - Take your stills

Here are some guidelines for taking your stills:

  • Keep the camera steady - use a tripod if available or get someone to take the photos for you.
  • You need about 5 stills for 1 sec of video - that means 50 for 10 secs!
  • Only change the objects' positions slightly for each still - it then gives a sense of movement.
  • Objects in funny positions work well in the video - play around!
  • Don't worry about the end result - you'll be surprised how well things look.

Step 3 - Copy stills onto your computer

Create a new folder (under Pictures) and minimise the window. Connect your camera and access the folder containing the stills just taken. Copy the stills into the folder on your computer.

Click below for more help on each of these steps.

Create a new folder on computer

  • Go to the Start Menu and select Pictures.
  • Under Pictures create a new folder.
  • Name the folder something related to your video e.g. games.
  • All files used for your video will be saved into this folder.
  • Minimise this window for now.

Connect camera and select the folder with stills

  • Connect the camera and select it from Start Menu - Computer
  • Locate the folder containing stills, usually locted under DCIM.
  • Select the folder (named something like 116_024). You'll copy to the PC in the next step.

Drag&Drop from camera to computer

  • Align the two windows side by side: one from the camera; one from the computer. These are the two windows you just minimised.
  • Remember to resize and move the windows so that they are both visible side by side.
  • Drag and drop the folder from the camera onto the computer.
  • This will copy all the stills contained in the folder on the camera onto your computer.
  • If you do not want to copy over the entire folder, just select the stills you want to copy. To select multiple stills: select the first one (Left-Click), hold the Shift key and select the last one. Once selected, drag and drop using the left mouse.

Step 4 - Make a basic video

Use Windows Movie Maker to make a basic video. Add the stills to a new project under Movie Maker, edit them to remove any unwanted stills, set the duration for each still to 0.2 secs and view results!

Click below for more help on each of these steps.

Start Movie Maker and add stills

  • Go to the Start Menu and type movie maker.
  • This will start up Windows Movie Maker in new window.
  • Add the stills by first selecting to Add videos and photos
  • Then browse to your folder (created under Pictures) and open it.
  • Select the first one (Left-Click), hold the Shift key, select the last one (Left-Click) and Open.
  • This copies all your photos into your new movie project. Don't worry, you'll have time to remove any unwanted photos shortly.

Save your project

It is a good idea to save your project now and remember to save regularly as you work.
To save:

  • Select to save your project.
  • Browse to save it to the folder you created on your computer (under Pictures) and name it something useful e.g. games.
  • This creates a project file for your work that you can return to at any time.

Remove any unwanted stills

  • Remove an unwanted stills from within Movie Maker.
  • Simply, select any stills your don't want to include (left-click) and then Right-Click – Remove.
  • Remember to save your work! (Ctrl-S will save it to the project file you just created)

Select all stills and change the duration

You are now ready to make the video.

  • Select the first one (Left-Click), hold the Shft key, select the last one (Left-Click)
  • With all selected, click Edit from the menu bar and set the duration to 0.2 secs.
  • Select Play to see how this looks.
    (On left hand side under Previewed photo)

Aside: You can change the duration of the display of any single still simply by just selecting that still and then modifying the duration. For this stop motion animation a duration of 0.2 secs for all stills creates a sense of motion.

Save your work and view it!

  • As before, save your work!
  • To make a playable video you must save your movie.
  • For this exercsie save your movie for email - this will make a small, manageable file.
  • As before, browse to save it to the correct folder.
  • You can now exit movie maker and view your created video.

Step 5 - Enhance your video

You may like to enhance your video by adding music, titles and credits. Select the options below to find out how.

Add music

  • First, select the music you wish to add. It is easiest if music is in mp3 format. Use music from:
    • Your own CDs (ripped onto your computer).
    • From mps3 files supplied (ask tutor).
    • YouTube - but remember music must be converted to mp3 format to use.
  • Copy this music into the same folder (under Pictures) as your video.
  • Add this music to your video using Movie Maker.
  • Once added the music can be editted to change:
    • Start Time i.e. when in your video the music will start at
    • Start Point i.e. the point in the music that will play
    • End Point i.e. the point in the music that the music will end (ignore this!)
  • The music can also be Faded in and Faded out. This has a nice effect at the start and the end of the video.

Add Titles

  • First position the cursor at the start of the movie i.e. just before the very first still.
  • Select title from the top banner. This will add a new slide containing text for your title.
  • Once added the text can be editted as can the duration of the title display.

Add Credits

  • To add credits position the cursor at the end of the movie i.e. just after the very last still.
  • Select credits from the top banner. This will add a new slide containing text for your credits.
  • As with the titles, the text can be editted as can the duration of the credit display.
  • Remember also that the music can be fadded in or out at this stage if not already done.

Step 6 - Share your video

Publish your video on YouTube. You can do this though your own account or through the account set up for this class.

  • First remember to save your project AND save your movie (save for email as this is quickest)
  • Sign in to YouTube using the name digitaloutputs2012. Your tutor will supply the password.
  • Select to upload the video.
  • Select the correct file from your computer, the file type is windows media video (.wmv).
  • You can add a Title, Description and Tag as you wish.
  • YouTube will generate a URL - click this to play your video.
  • Email this URL to yourself for future reference!
